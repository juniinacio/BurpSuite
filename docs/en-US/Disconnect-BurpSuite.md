# Disconnect-BurpSuite

## SYNOPSIS
Removes current BurpSuite session.

## SYNTAX

```
Disconnect-BurpSuite [<CommonParameters>]
```

## DESCRIPTION
Removes current BurpSuite session.

## EXAMPLES

### Example 1
```powershell
PS C:\> Disconnect-BurpSuite
```

This example show how to disconnect the current BurpSuite session.
